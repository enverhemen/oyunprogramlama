# oyunprogramlama
Örnek Kodlarım [Unity] [C#]

Android ve iOS platformları için hazırlamış olduğum bölüm serisi halinde level mantığı ile ilerleyen Unity3d oyun motoru arayüzü ile hazırladığım 2D fiziğe ve grafiklere sahip mobil oyunumdur.

* Oyunumda C# diliyle kodlama yaptım.
* Adobe Photoshop CC 2019 ve Adobe Illustrator kullanarak grafik ve logo tasarımını yaptım.
* Arayüz tasarımı, Logo tasarımı, Cisim ve Engeller, Destekler tasarımlarını yaptım.
* Oyun içi Animasyonları Unity 3d arayüzü ile hazırladım.
